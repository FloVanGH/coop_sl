## `TabItem`

Represents an item in a TabView.

### Fields

-   **`icon`** (_image_): Icon that is displayed in front of the item.
-   **`text`** (_text_): The header text of the item.
-   **`closeable`** (_bool_): If set to true the item can be closed.

## `Icography`

Size settings for icons.

### Properties

-   **`small`** (_out_ _length_): The size for small icons (16px).
-   **`medium`** (_out_ _duration_): The size for medium sized icons (18px).
-   **`large`** (_out_ _duration_):  The size for large icons (20px).
